# AI Energy Platform

AI Energy Platform is a modular SaaS foundation for intelligent energy management spanning solar, second-life EV batteries, home and industrial storage, predictive maintenance, and AI-driven optimization.

## Architecture overview

- Frontend: React, TypeScript, Vite, TailwindCSS, React Router, React Query, Framer Motion
- Design: premium dark UI with responsive dashboards and reusable components
- PWA: installable shell with offline caching and manifest metadata
- Backend foundation: production-oriented folder structure prepared for FastAPI, PostgreSQL, Redis, MQTT, and Celery

## Repository structure

- frontend/: future web application shell
- backend/: API and service layer foundation
- docs/: architecture notes and implementation plans
- .github/workflows/: CI/CD pipeline scaffold

## Getting started

1. Install dependencies with npm install
2. Start the development server with npm run dev
3. Build for production with npm run build
